MAME FILE MANAGER User Instructions

Settings View

MFM on the first run takes you to the Settings View. You enter your MAME exe and directories information. You can change these at any time by selecting MFM -> Settings

  1. Select your MAME exe
  2. Select your full ROMs set directory . Can be a parent or ancester directory but doing this will increase search times
  3. Select your full CHDs set directory, if different from ROMs
  4. Select your full EXTRAs set directory
  5. Select your Videos set directory
  6. Select parent directory where you want your PLAYABLE game resources placed
    • If MFM does not find needed directories in the MAME Play directory it will create them i.e. "artwork", "bkground", "cabinets", "chd", "cpanel", "flyers", "icons", "manuals", "marquees", "pcb", "roms", "samples", "snap", "titles"
    • WARNING ! MFM will overwrite existing files in those directories - this should not be an issue but if you have older version files you want to keep please use a new directory for MFM to manage

When you save your settings the first time MFM will run MAME to get all games information for this version of MAME. NOTE this may take a few minutes depending on your system. When completed you will be brought to the main view with a single list of all PLAYABLE games for this MAME version. This is the Game List view

Game List View

  • Double click on a game and if it has been populated(or your MAME.ini points to your full set resources also), see Creating Lists below, MAME will be started with that game
  • Press ENTER will also play a game with the same caveat
  • Use right mouse to get a popup with selections to:
    • Add or Remove selected game to a list
    • Play and Record selected game(INP)
    • Playback a recorded game and create an AVI
    • Edit AVI - opens into VDub
    • Display the History, MAMEInfo or Manual for this game if you have it
    • Play the Video of the game if you have one
    • Exit the application
  • Press F1 will show selected game's History
  • Press F2 will show selected game's Manual
  • Press F3 will show selected game's MAMEInfo
  • Press F4 will Start selected game and Record to .INP file
  • Press F5 will Playback and record to AVI of selected game(if .inp file exists)
  • Press F6 will Playback selected game from .INP
  • F7 unassigned
  • Press F8 will Play Video of selected game
  • Press 'a' switches to ALL list
  • Press 'c' switches to CLONE list
  • Press 'n' switches to NOCLONE list
  • Press 'p' switches to PD VIDs list
  • Press 'v' switches to VERTICAL list

Create List View

List -> Create List

In the right hand panel you can type in Game names. Click the Pick Games button at the top to open a game picker to select games. In the picker you can double click on a game name to add it to the list. Or select the game and use the Add button. The picker has a basic lookup capability. Start to type in a name and it will traverse to it. At the bottom of the panel enter the name of this list. When you are finished click Create Play List

LIST -> MyLists

By default includes built in lists: ALL, CLONE, NOCLONE, PD VIDs, VERTICAL

List -> Populate Populate a list. This will search for all associated files for this game and copy them to your play directories. While populating a small red clock arm will be visible at the window bottom center of the statusbar.

List -> Save List to File Saves selected list game names to a text file in <MFM Root>/Lists


NOTE : until you have populated a list or manually moved the ROMs/CHDs the games will not be playable from MFM if you have the standard mame.ini setup

Command View

Command -> MAME Command Builder

Lists all of the MAME commands in a dialog. Double click and the command will be added to the textfield. For new commands to be saved add entry to drop down box and click Save. Run executes the command currently displayed in the textfield.

Command window output appear in the MAME_OUTPUT log.

NOTE piping command output to file is not supported in 0.5 release - we are calling MAME without a command shell

Font Size and Look & Feel

UI -> Font Size Increases/Dedreases the font size. NOTE: this is done by a very basic HACK. Java does not provide a universal UI font setting. So it is hit or miss and if you change the Look & Feel it will get partially or completely overridden.

UI -> Look & Feel Select any of the available L&F for your platform. NOTE: you will find with certain L&Fs that there will be 'bugs' in the layout and function of the UI. Coding and testing for every L&F is beyond the current scope of this project.

0.4 adds support for: JGoodies http://www.jgoodies.com/ - SeaGlass https://seaglass.googlecode.com - JTattoo http://www.jtattoo.net/index.html

Place jars into <MFM Root>/JARs

Jars\seaglasslookandfeel-0.2.jar

Jars\jgoodies-looks-2.6.0.jar

Jars\jgoodies-common-1.7.0.jar

Jars\JTattoo-1.6.10.jar

Video Menu

VDub & FFmpeg are available based on the PD Vids project needs. You must have them properly installed and the FFmpeg project version.

Video -> VDub opens VDub. If the selected game in the list has an AVI file in your MFM configured video sets directory that file will open into VDub.

Video -> FFmpeg opens a dialog to set or confirm FFmpeg exe location and your input(AVI files) and output(MP4) directories.

Convert Files button runs FFmpeg. DO NOT close MFM during this operation. MFM runs FFmpeg file by file. FFmpeg output is captured in <MFM Root>/Logs/FFmpeg_OUT--<number>.txt

Video -> Extract GIF Images WARNING ! Memory hog. Increase memory to at least -Xmx384m

GIF images are saved to <MFM Root>/Images/<gamename>-Frame<#>.png
