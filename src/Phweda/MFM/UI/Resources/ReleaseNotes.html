MFM Release Notes

0.1 Jan 7th 2012

Alpha

0.2 Jan 20th 2012

Beta with full basic functionality

0.3 April 2012

Windows full release

  • Added error message popup if MAME fails when run from MFM. The MAME error message that you would see when run in a command window is captured and displayed.
  • Several major code refactors primarily for maintainability and clarity.

0.4 MARCH 2015

  • Added logs menu.
  • Added numerous functions/hot keys see Usage.
  • Started adding Video capture functions such as Record Game to .inp
  • Added 4 built in Lists.
  • Added new Look and Feels -
    • JGoodies http://www.jgoodies.com/
    • SeaGlass https://seaglass.googlecode.com
    • JTattoo http://www.jtattoo.net/index.html

0.5 MAY 2015

  • Added full support for the PD Videos project
  • Video menu
    • VDub - opens VDub and if selected game has a .avi opens that into VDub
    • FFmpeg - runs .avi to .mp4 conversion on selected folder
    • Extract GIF Images - extracts images from selected GIF file for viewing and saving
  • Right click popup menu includes
    • Record Game - to .inp
    • Playback to AVI - record and save to .avi
    • Play Video - if it exists in 'snap' folder, opens via OS to your player.
    • Edit Video - opens .avi inot VDub for editing
    • Game Video Info - displays game's size and orientation (from MAME) and the MFM calculated snapsize for recording
  • A few other tweaks/fixes/additions
